# API_MUTANT
API para verificar si un ADN enviado corresponde a un mutante o un humano. Adicional a lo anterior tenemos la posibilidad de consultar las estadísticas de los ADN analizados.
# Tecnologías usadas
- Java jdk-11
- Spring Boot
- Gradle
- MySQL
- Swagger
# Arquitectura
